# Calculator

A simple native iOS calculator app built in Swift using Storyboard-based UI.

## Overview

This project is a small calculator application that performs basic arithmetic operations. It is designed as a straightforward learning project and demonstrates how to build a simple iOS app using UIKit and Swift.

## Functionality

- Add and subtract numbers
- Enter multi-digit values and decimal numbers
- Clear the current input
- Simple user interface for quick calculations

## Project Structure

- `calculator/`
  - `AppDelegate.swift` — app lifecycle management
  - `ViewController.swift` — main calculator logic and user interaction
  - `Info.plist` — app metadata and configuration
  - `Main.storyboard` — user interface layout
- `utlities/`
  - `OperationsEnum.swift` — operation types used in the calculator logic

## How to Run

1. Open `calculator.xcodeproj` in Xcode.
2. Select a simulator or connected device.
3. Build and run the app.
